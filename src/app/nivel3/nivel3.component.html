Impresión y Variables

🖨️ ¿Qué es imprimir en pantalla? 🖨️

Imprimir no significa usar una impresora. En programación, imprimir significa mostrar algo en la pantalla. Para hacerlo, usamos printf.

Por ejemplo:

printf("¡Hola mundo!");

👉 Muestra: ¡Hola mundo!

Todo lo que esté entre comillas ("...") es lo que se verá.

3. Variables

Las variables las vimos en el nivel 2. Estas variables guardan valores, y se puede imprimir por pantalla también lo que guarda una variable.


Notacion: printf(nombre_de_la_variable)

MUY IMPORTANTE! Las variables no van entre " ".


📘 Escribe: 📘
