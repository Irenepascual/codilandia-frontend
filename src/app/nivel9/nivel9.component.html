Bucles while

🔁 ¡Repetimos hasta que cambie algo!

A veces, queremos que un programa haga algo muchas veces, ¡hasta que ya no sea necesario!

Por ejemplo:
👉 Mientras no esté limpia tu habitación, sigues ordenando. 🧹
👉 Mientras no te acabes el helado, sigues comiendo. 🍦

Para eso usamos while (que significa "mientras").

Así le decimos al programa: "Mientras pase algo, sigue repitiendo esto."

¡Usemos variables booleanas!

Podemos usar variables booleanas para evaluar nuestros bucles while. El bucle terminará cuando el valor de la variable sea true o false según nos convenga. 

boolean acabado = false;
  while (acabado == false) {{ '{' }}
     // Espero
  {{ '}' }}

⚠️ ¡Cuidado con los bucles infinitos!

A veces, si olvidamos cambiar la condición, el bucle nunca termina. Eso se llama un bucle infinito y puede hacer que el programa se quede atascado. 😱

Por ejemplo: 
boolean jugando = true;
      while (jugando == true) {{ '{' }}
         juego();
      {{ '}' }}

Si nunca ponemos jugando = false; dentro del bucle, ¡nunca parará de jugar! 🎮

⚠️ ¡Recuerda siempre asegurarte de que el bucle pueda terminar!
