Variables y Tipos de Datos

🧠 ¿Qué es una variable? 🧠

Una variable es como una cajita donde guardamos información en un programa. Le damos un nombre, y dentro metemos un valor. Las variables se usan mucho para que el ordenador recuerde datos como números, frases, o si algo es verdadero o falso.

✍️ En programación, las variables se escriben así:

tipo nombre_variable = valor;

Es importante que el nombre de una variable represente lo que se quiere guardar (ya que así es más fácil recordar luego😉).

Una variable siempre debe terminar en ";".

Por ejemplo: int edad = 9;

1. Entero

¿Qué es un entero?

Es un número sin decimales, como 1, 2, 3... ¡como contar manzanas!

Ejemplo: Si escribes 4, verás 4 manzanas aparecer 🍎🍎🍎🍎


Escribe un número del 1 al 10. Mostraremos tantas manzanas como número elijas.

Notación: int numero_manzanas

✅ ❌ int numero_manzanas = {{ numeroManzanas }};

¡Mira esta imagen!

¿Cómo escribirías esa variable en código para ver cuántos plátanos hay?

Ejemplo notación: int numero_platanos = 100;

✅ ❌ int numero_platanos = 4;

2. Real

¿Qué es un número real?

Es un número que puede tener decimales, como 1.50 o 2.55.

Ejemplo: Si usas una moneda de 1 €, otra de 1 € y una de 0.55 €, juntas hacen 2.55 €.


Selecciona monedas para sumar 2,55 €.

Notación: double monedas

✖

Total: {{ sumaMonedas.toFixed(2) }} €

✅ ❌ double monedas = {{ sumaMonedas.toFixed(2) }};

3. Booleano

¿Qué es un booleano?

Es una variable que solo puede tener dos valores: true (verdadero) o false (falso).

Ejemplo: ¿Hoy he desayunado? Si sí, entonces es true, si no, es false.

Notación: bool he_desayunado = true;


¿Hay un pastel?

✅ ❌ bool hay_pastel = true;



¿Hay tres trozos de pastel?

✅ ❌ bool hay_tres_trozos = false;

4. String

¿Qué es un string?

Es un tipo de variable que se usa para guardar palabras o frases. Como cuando escribes tu nombre o una historia.

Ejemplo: "Hola, me llamo Leo"

Notación: string texto


Escribe tu frase favorita:

✅ ❌ string texto = "{{ textoUsuario }}";

Escribe esta frase exactamente:

"Tres tristes tigres comían trigo en un trigal"

Notación: string frase = "...";


✅ ❌ string frase = "Tres tristes tigres comían trigo en un trigal";
